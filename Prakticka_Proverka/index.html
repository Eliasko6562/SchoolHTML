Adresy používané na internetu

URL adresa je kompletní zápis www stránky, který najdete ve vyhledávacím řádku.
Krom onoho počátečního https:// (což je bezpečnostní http protokol) a www je součástí
URL název domény a také adresa konkrétní podstránky, uvedená za lomítkem.
Tento článek má URL adresu: www.webglobe.cz/blog/co-je-domena.

Seznam významných termínů a zkratek

HTML
Jazyk, který se používá k vytváření obsahu webové stránky.
HTTP
Slouží pro přenos hypertextových dokumentů ve formátu HTML, XML, i jiných typů souborů.
Validátor
automatický nástroj umožňující ověřit validitu dokumentu, tj. najít případné chyby v zápisu kódu jeho porovnáním s platnými standardy.
Server
označení pro počítač, který poskytuje nějaké služby, nebo počítačový program, který tyto služby realizuje.
